# Beautiful [http://bottlepy.org/](Bottle) Skeleton

Because there aren't enough Bottle skeleton web applications already in existence that look beautiful.

## Setup

* `pip install -r requirements.txt`

## Running

* `python webapp.py`
* Browse to localhost:8080
